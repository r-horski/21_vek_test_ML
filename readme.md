# Соревнование: Предсказание ключевого фактора спроса

В рамках данного соревнования вам предстоит построить систему, которая будет предсказывать один из ключевых факторов, характеризующих спрос на товары. Этот фактор играет важную роль в принятии решений в таких областях, как:

- Оптимизация ценовой стратегии  
- Маркетинговые кампании  
- Управление складскими запасами  

для компании **21vek.by**.

## Оценка результатов

Качество моделей оценивается по метрике **F1 (binary)** — гармоническое среднее между точностью и полнотой для бинарной классификации.

## Формат файла с ответами

Файл с предсказаниями должен содержать заголовок и быть представлен в формате CSV. Каждая строка (после заголовка) должна содержать `id` объекта и соответствующее предсказанное значение `y`.

Пример корректного файла:
id,y
2,0
5,0
6,0
etc.

Убедитесь, что:
- Все `id` из тестовой выборки присутствуют.
- Предсказания являются бинарными (только `0` или `1`).
- Файл имеет расширение `.csv`.
