Dataset Description

Таргет и признаки в рамках данного соревнования анонимизированы в следствии не этичности разглашения бизнес метрик компании 21vek.
Файлы

    train.csv - тренировочный датасет
    test.csv - тестовый датасет
    sample_submission.csv - пример правильно сформированного файла с предсказаниями

Краткое описание данных и таргета:

    id - уникальный идентификатор строки
    relative_date_number - время относительно некой даты в прошлом (для каждого g1, g2 в train/test меет единый фиксированный диапазон без пропусков)
    g1, g2 - анонимизированные id групп по иерархии каталога
    y - фактор, который необходимо предсказать
    x1-x12 - анонимизированные фичи
